# 💎 宝石消消乐 - 三消游戏

一个功能丰富、界面精美的网页版三消游戏,包含多种宝物类型、特殊道具和游戏模式。

## 🎮 游戏特色

### 多种宝物类型
- **8种不同宝石**: 红宝石💎、蓝心💙、绿心💚、星星⭐、水晶球🔮、橙心🧡、粉心💗、钻石💠
- **特殊宝石**:
  - 横向消除宝石 (连成4个)
  - 纵向消除宝石 (连成4个)
  - 彩虹宝石 (连成5个或以上)

### 三种游戏模式

#### 🎮 经典模式
- 无限移动次数
- 挑战最高分数
- 适合休闲娱乐

#### ⏱️ 限时模式
- 60秒倒计时
- 紧张刺激的快节奏游戏
- 考验反应速度

#### 🎯 关卡模式
- 每关30次移动机会
- 达到目标分数即可过关
- 关卡难度递增
- 过关奖励额外道具

### 🎁 强大的道具系统

1. **💣 炸弹** (初始3个)
   - 消除3x3区域的所有宝石
   - 快速清理棋盘

2. **🔄 重排** (初始2个)
   - 重新排列整个棋盘
   - 解决无可消除的困境

3. **💡 提示** (初始5个)
   - 显示可消除的宝石组合
   - 帮助找到最佳移动

4. **🌈 彩虹炸弹** (初始1个)
   - 消除所有同色宝石
   - 超强威力道具

### 🌟 游戏机制

- **连击系统**: 连续消除会获得额外分数奖励
- **特殊宝石生成**:
  - 4个宝石连成: 生成横向/纵向消除宝石
  - 5个或以上: 生成彩虹宝石
- **流畅动画**: 消除、下落、填充动画效果
- **智能匹配检测**: 自动检测并消除匹配的宝石

## 🚀 如何开始

### 在线游玩

1. 直接用浏览器打开 `index.html` 文件
2. 选择游戏模式开始游戏

### 本地部署

```bash
# 克隆或下载项目文件
# 确保以下文件在同一目录:
# - index.html
# - style.css
# - game.js

# 使用任意HTTP服务器运行 (可选)
python -m http.server 8000
# 或
npx serve
```

然后在浏览器中访问 `http://localhost:8000`

## 🎯 游戏规则

### 基本玩法
1. 点击选择一个宝石
2. 再点击相邻的宝石进行交换
3. 形成3个或以上相同宝石即可消除
4. 消除后上方宝石会下落,新宝石会填充空位

### 得分规则
- 每消除一个宝石: 10分
- 连击奖励: 连击数 × 5分
- 特殊宝石消除: 20分/个
- 4个连成: +50分奖励
- 5个连成: +100分奖励

### 关卡模式特殊规则
- 初始30次移动
- 达到目标分数即可进入下一关
- 每关目标分数提高50%
- 过关奖励:
  - +1个炸弹道具
  - +2个提示道具
  - 重置移动次数

## 🔧 游戏配置

可以在 `game.js` 中修改游戏参数:

```javascript
const BOARD_SIZE = 8;        // 棋盘大小 (默认8x8)
const GEM_TYPES = [...];     // 宝石类型
gameState.timer = 60;        // 限时模式时间 (秒)
gameState.moves = 30;        // 关卡模式移动次数
gameState.targetScore = 1000; // 关卡模式初始目标分数
```
